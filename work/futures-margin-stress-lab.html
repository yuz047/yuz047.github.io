Margin rate file

Upload IR/MR/intraday rates separately from positions. Default columns preserve the baseline rate; editable columns override the default only when populated. Blank editable cells fall back to default.

Rate CSV schema: symbol, product, defaultInitialRate, initialRate, defaultMaintenanceRate, maintenanceRate, defaultIntradayRate, intradayRate, source. Symbol can be blank for product-level fallback. Editable columns can be blank to use default.

Position book

Position rows carry exposure only. Effective rates are joined from the uploaded rate book by symbol first, then product fallback.

Position CSV schema: symbol, product, side, qty, price, multiplier. Side accepts Long/Short or signed quantity.
