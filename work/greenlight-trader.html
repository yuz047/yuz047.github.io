Greenlight Trader 2.0

A Massive-first systematic PM agent for a $5,000 paper portfolio. It trains on historical market data, replays the same daily loop used for live paper trading, learns constrained score weights, rotates ETFs dynamically, compares against simple benchmarks, and watermarks every generated output.

Core idea

A trading agent that does less, but records more.

Greenlight is not a free-form agent that invents trades. The production track is deterministic: build a daily universe, compute features, score stocks and ETFs, select dynamic ETF exposure, allocate target weights, run risk gates, and trade only when drift, persistence, interval, turnover, and minimum-size rules agree. The AI layer reviews and explains; it cannot execute or override risk.
