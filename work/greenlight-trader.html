What it does

From market regime to target allocation.

Free OHLCV from yfinance plus volatility and benchmark signals feed a daily allocation engine. The strategy decides what the next dollar deserves: SPY ballast, QQQ, semiconductor beta through SMH, selective Mag7 and chip sleeves when the setup is clean, SHY in defensive regimes, or cash when VIX is high and better dip entries may arrive. The book does not need to spend all buying power in one trade. Every run writes target weights, live positions, risk state, and an EOD review for audit.

Architecture

Engine writes, dashboard reads.

One source of truth — the Python engine — and a public read-only dashboard. The cron runs daily on a GitHub Actions runner; it pulls fresh data, recomputes signals, advances the book, writes JSON snapshots to the repo, persists the paper-book state, and (optionally) mirrors them into a Supabase Postgres database. The dashboard is a static page that fetches those snapshots over HTTPS.

Hot-swappable logic

Strategies are files, not branches.

Every strategy lives in its own module under python/strategies/. Each file exposes a manifest (id, version, rules, parameters) and either a target_weights() allocator or a run() pitcher entrypoint. On startup the package auto-discovers every module — no central register to edit. To activate a new strategy, list its id in config.ACTIVE_IDS. To upgrade, drop a file with a bumped version next to the old one and switch the id. Old versions stay readable for audit and back-comparison; nothing mutates rules at runtime.

Why the friction is the feature

Friction stops the system from chasing today's losing trade with tomorrow's new rule. The EOD reviewer can propose a change (logged in the decision log below), but it cannot apply one. A human runs a comparison backtest, accepts or rejects, and bumps the version. This is how strategy drift is controlled.

Risk framework

Relative to SPY, not absolute.

The newer risk gate starts with allocation. SPY remains the benchmark, but the book can lean into QQQ and semiconductors in risk-on regimes, throttle single-name sleeves, move to SHY when trend breaks, and leave cash when VIX says patience is worth more than exposure. The mandate is still to beat SPY by 10%, but the relative drawdown budget is now 8% and the default single-stock sleeve is capped around 10% of NAV.
